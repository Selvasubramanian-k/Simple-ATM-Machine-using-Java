# ATM Machine Java Program

## Description

This is a simple Java program simulating an Automated Teller Machine (ATM). The application allows users to perform basic ATM operations such as checking their balance, withdrawing funds, depositing money, and exiting the program. It requires a 4-digit password for authentication.

## Features

- User Authentication: Verifies that the entered password is a 4-digit number.
- Balance Management: Allows users to check their current balance.
- Withdraw Funds: Users can withdraw money if their balance is sufficient.
- Deposit Funds: Users can deposit money into their account.
- Exit Option: Users can exit the application.
